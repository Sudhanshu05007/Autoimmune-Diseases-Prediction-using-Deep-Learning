* Medical diagnosis analysis and prediction framework using clinical dataset
* Includes data loading, preprocessing, and missing value inspection
* Performs exploratory data analysis (EDA) with statistical visualizations
* Encodes categorical features such as gender and diagnosis labels
* Uses MultiLabelBinarizer for multi-class diagnosis preparation
* Generates plots like histograms, count plots, scatter plots, and correlation heatmaps
* Built with Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn
* Implemented in Jupyter Notebook for healthcare data analysis and model readiness
